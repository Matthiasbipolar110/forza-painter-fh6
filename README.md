6. Click Save to Project.
7. Open Forza Horizon 6. 
8. Go to the design editor. Your imported vinyl exists in your library under the name you provided.

## 🔧 Troubleshooting Common Issues

Errors occur when file paths change or game updates modify folder structures. Follow these methods to fix basic problems:

### The tool does not open
Check if your antivirus software flagged the program. Some security tools mistake custom software for threats. Add the folder to your exclusion list. Ensure you extracted the files from the zip folder before you attempt to run them.

### Designs do not appear in the game
Confirm that you saved the imported file to the correct folder. Open the settings menu inside the painter tool. Verify the path matches your game installation folder exactly. Close and restart the game after you save new designs to force the game to refresh the library.

### The image quality looks poor
Forza handles vinyls using paths and shapes. Images with many colors or gradients cause issues. Use high-contrast images with clean lines. The converter works best with black and white shapes. You can modify the threshold and detail sliders in the import window to improve the result.
